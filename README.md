# Hit Master

## Gameplay :video_game:
![Game Preview](GitHub/HitMaster-gameplay.gif)

### Как управлять
Нажимай на место, в которое будет стрелять персонаж.

### Цель
Стрелять по ботам, чтобы идти дальше.

## Требования :traffic_light:
